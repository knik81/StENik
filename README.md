# StENik
Мобильное андроид приложение для изучения иностранных языков. 
## Содержание
- [Функции](#функции)
- [Технологии](#технологии)

# Функции:
-  Чтение слов и фраз на разных языках. Фоновая работа.
- Хранение базы слов в приложении на телефоне, в Firebase. Загрузка базы из файла Excel, ручной и голосовой ввод.
- Функция тестирования знаний.
- Возможность настройки приложения: Языки, скорость чтения, принцип выбора слов и фраз и др.

# Технологии
Kotlin, Jetpack Compose, MVVM, Clean Architecture с разделением на модули, Coin, JSON, ROOM, SharedPreferences, Coroutines, Navigation, Glide, Firebase(Firestore, Authentication).
